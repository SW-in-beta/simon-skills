# Wireframes: InstaClone

```
┌─────────────────────────────────┐
│                                 │
│         InstaClone              │
│                                 │
│   ┌─────────────────────┐      │
│   │ Email               │      │
│   └─────────────────────┘      │
│   ┌─────────────────────┐      │
│   │ Password            │      │
│   └─────────────────────┘      │
│                                 │
│   [      로그인       ]         │
│                                 │
│   계정이 없으신가요? 가입하기    │
│                                 │
└─────────────────────────────────┘
```

│  │                          │   │
