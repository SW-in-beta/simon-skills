## Walking Skeleton (Sprint 1 목표)

핵심 흐름이 끝에서 끝까지 동작하는 최소 버전:

```
행동 이벤트 수집(US-1) → 상품 등록(US-2) → 모델 학습(US-3) → 추천 API 제공(US-4)
```

이 Walking Skeleton이 완성되면, 사용자가:
1. 상품을 등록하고
2. 사용자 행동 데이터를 수집하고
3. 추천 모델을 학습하고
4. 추천 API로 개인화된 추천을 받을 수 있다
