# Phase 1 산출물 요약: SnapShare (인스타그램형 소셜 미디어 웹앱)

인스타그램 스타일의 소셜 미디어 웹 애플리케이션(SnapShare)의 Phase 1 기획 산출물입니다. 사용자가 사진을 올리고, 팔로우하고, 좋아요/댓글을 달 수 있으며, 최신순 피드와 알림 기능을 제공합니다.

## 핵심 기술 결정 사항

| 결정 항목 | 선택 | 근거 |
|-----------|------|------|
| 프레임워크 | Next.js 14+ (App Router) | SSR, API Routes 통합, Vercel 배포 최적화 |
| UI | Tailwind CSS + shadcn/ui | 빠른 개발, 일관된 디자인 시스템 |
| DB | PostgreSQL 16 | ACID 보장, 풍부한 인덱스, trgm 검색 |
| 캐시 | Redis | 세션, 알림 카운터, Rate Limiting |
| ORM | Prisma | 타입 안전, 마이그레이션, 코드 생성 |
| 인증 | JWT (access + refresh) | 무상태, 확장성 |
| 이미지 | S3 + CloudFront CDN | 확장성, 비용 효율 |
| 피드 | 최신순 SQL 쿼리 | MVP 복잡도 최소화 |
| 알림 | 폴링 (30초) | MVP 복잡도 최소화, 향후 WebSocket 전환 |
