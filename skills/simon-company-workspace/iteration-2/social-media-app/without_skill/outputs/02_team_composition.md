# 팀 구성 및 역할 정의

## 1. 조직 구조

```
프로젝트 매니저 (PM)
├── 프론트엔드 팀
│   ├── 시니어 프론트엔드 개발자 (테크 리드)
│   ├── 프론트엔드 개발자 1
│   └── 프론트엔드 개발자 2
├── 백엔드 팀
│   ├── 시니어 백엔드 개발자 (테크 리드)
│   ├── 백엔드 개발자 1
│   └── 백엔드 개발자 2
├── 디자인 팀
│   ├── UI/UX 디자이너
│   └── 그래픽 디자이너
├── QA 팀
│   └── QA 엔지니어
└── DevOps
    └── DevOps 엔지니어
```

## 2. 역할별 상세 정의

### 프로젝트 매니저 (PM) - 1명
**책임:**
- 프로젝트 전체 일정 및 리소스 관리
- 스프린트 계획 및 회고 진행
- 이해관계자 커뮤니케이션
- 리스크 관리 및 이슈 에스컬레이션
- 요구사항 우선순위 조정

**필요 역량:**
- 애자일/스크럼 경험 3년 이상
- 소셜 미디어 또는 B2C 서비스 프로젝트 경험
- JIRA, Confluence 등 프로젝트 관리 도구 숙련

---

### 시니어 프론트엔드 개발자 (테크 리드) - 1명
**책임:**
- 프론트엔드 아키텍처 설계 및 기술 의사결정
- 코드 리뷰 및 코딩 표준 수립
- 성능 최적화 (이미지 로딩, 무한 스크롤 등)
- 프론트엔드 팀 멘토링

**필요 역량:**
- React/Next.js 5년 이상 경험
- 대규모 이미지 처리 웹앱 경험
- TypeScript, Tailwind CSS 숙련
- 웹 성능 최적화 전문 지식

### 프론트엔드 개발자 - 2명
**책임:**
- UI 컴포넌트 개발 (피드, 프로필, 업로드 등)
- 반응형 웹 디자인 구현
- API 연동 및 상태 관리
- 단위 테스트 및 통합 테스트 작성

**필요 역량:**
- React/Next.js 2년 이상 경험
- TypeScript 숙련
- 반응형 웹 개발 경험
- 테스트 프레임워크 (Jest, Testing Library) 경험

---

### 시니어 백엔드 개발자 (테크 리드) - 1명
**책임:**
- 백엔드 아키텍처 설계 (API, DB 스키마, 인증)
- 알림 시스템 설계
- 이미지 처리 파이프라인 설계
- 보안 및 성능 최적화
- 코드 리뷰 및 백엔드 팀 멘토링

**필요 역량:**
- Node.js/TypeScript 5년 이상 경험
- PostgreSQL, Redis 설계 및 운영 경험
- AWS S3 등 클라우드 스토리지 경험
- 인증/인가 시스템 설계 경험
- 대규모 트래픽 처리 경험

### 백엔드 개발자 - 2명
**책임:**
- REST API 엔드포인트 개발
- 데이터베이스 마이그레이션 및 쿼리 최적화
- 비즈니스 로직 구현
- 단위 테스트 및 통합 테스트 작성

**필요 역량:**
- Node.js/TypeScript 2년 이상 경험
- PostgreSQL 및 ORM(Prisma) 경험
- RESTful API 설계 경험
- TDD 경험

---

### UI/UX 디자이너 - 1명
**책임:**
- 사용자 리서치 및 페르소나 정의
- 와이어프레임 및 프로토타입 제작
- UI 디자인 시스템 구축
- 사용성 테스트 설계 및 진행

**필요 역량:**
- UI/UX 디자인 3년 이상 경험
- Figma 숙련
- 소셜 미디어 앱 디자인 경험 우대
- 디자인 시스템 구축 경험

### 그래픽 디자이너 - 1명
**책임:**
- 아이콘, 일러스트레이션 제작
- 브랜드 아이덴티티 수립
- 마케팅 에셋 제작

**필요 역량:**
- 그래픽 디자인 2년 이상 경험
- Adobe Creative Suite 또는 Figma 숙련

---

### QA 엔지니어 - 1명
**책임:**
- 테스트 전략 수립 및 테스트 케이스 작성
- 수동 및 자동화 테스트 수행
- 버그 리포트 및 회귀 테스트
- E2E 테스트 자동화 (Playwright)

**필요 역량:**
- QA 3년 이상 경험
- 테스트 자동화 도구 경험 (Playwright, Cypress)
- API 테스트 도구 경험 (Postman, k6)

---

### DevOps 엔지니어 - 1명
**책임:**
- CI/CD 파이프라인 구축 (GitHub Actions)
- 클라우드 인프라 구성 및 관리
- 모니터링 및 알럿 시스템 구축
- 배포 자동화

**필요 역량:**
- AWS 또는 유사 클라우드 3년 이상 경험
- Docker, Kubernetes 경험
- CI/CD 파이프라인 구축 경험
- IaC (Terraform 등) 경험
