# 팀 구성 및 역할 정의

## 1. 프로젝트 매니저 (PM) - 1명

- 프로젝트 일정 및 마일스톤 관리
- 이해관계자 커뮤니케이션
- 리스크 관리 및 의사결정
- 스프린트 계획 및 회고 주도

## 2. 프로덕트 디자이너 (UX/UI) - 1명

- 사용자 리서치 및 페르소나 정의
- 와이어프레임 및 프로토타입 제작
- UI 디자인 시스템 구축 (컴포넌트, 컬러, 타이포그래피)
- 사용성 테스트 설계 및 수행

## 3. 프론트엔드 개발자 - 2명

### 리드 프론트엔드 (1명)
- 프론트엔드 아키텍처 설계 (Next.js + React)
- 공통 컴포넌트 및 디자인 시스템 구현
- 상태 관리 전략 수립
- 코드 리뷰 및 기술 의사결정

### 프론트엔드 개발자 (1명)
- 피처 개발 (피드, 프로필, 알림 등)
- 반응형 UI 구현
- 이미지 업로드/표시 최적화
- E2E 테스트 작성

## 4. 백엔드 개발자 - 2명

### 리드 백엔드 (1명)
- 백엔드 아키텍처 설계 (API, DB 스키마)
- 인증/인가 시스템 구현
- 피드 알고리즘 및 알림 시스템 설계
- 성능 최적화 및 코드 리뷰

### 백엔드 개발자 (1명)
- API 엔드포인트 구현
- 이미지 저장/처리 파이프라인 구현
- 알림 시스템 구현
- 단위/통합 테스트 작성

## 5. DevOps / 인프라 엔지니어 - 1명 (파트타임 또는 겸임)

- CI/CD 파이프라인 구축
- 클라우드 인프라 설정 (AWS/GCP)
- 이미지 스토리지(S3/CloudStorage) 및 CDN 설정
- 모니터링 및 로깅 시스템 구축

## 6. QA 엔지니어 - 1명 (파트타임 또는 겸임)

- 테스트 전략 수립
- 기능 테스트 및 회귀 테스트 수행
- 버그 트래킹 및 품질 보고
- 성능/부하 테스트

## 기술 스택 (참고)

| 영역 | 기술 |
|------|------|
| 프론트엔드 | Next.js, React, Tailwind CSS, shadcn/ui |
| 백엔드 | Node.js (Express 또는 Fastify) 또는 Go |
| 데이터베이스 | PostgreSQL (주 DB), Redis (캐시/세션) |
| 이미지 저장 | AWS S3 + CloudFront CDN |
| 인증 | JWT + OAuth 2.0 (소셜 로그인) |
| 실시간 알림 | WebSocket (Socket.io) 또는 SSE |
| 인프라 | Docker, GitHub Actions (CI/CD) |
