# UI/UX 와이어프레임 명세

## 페이지 목록

| 페이지 | 경로 | 인증 | 설명 |
|--------|------|------|------|
| 로그인 | /login | 불필요 | 이메일/비밀번호 로그인 |
| 회원가입 | /register | 불필요 | 신규 계정 생성 |
| 피드 | / (또는 /feed) | 필요 | 메인 피드 (최신순) |
| 게시물 상세 | /post/[id] | 필요 | 게시물 + 댓글 |
| 업로드 | /upload | 필요 | 새 게시물 작성 |
| 프로필 | /profile/[username] | 필요 | 사용자 프로필 + 게시물 그리드 |
| 프로필 수정 | /settings/profile | 필요 | 내 프로필 편집 |
| 알림 | /notifications | 필요 | 알림 목록 |

### 피드 카드 상세 동작
- 이미지 더블 탭/클릭: 좋아요 토글 (하트 애니메이션)

- username 클릭: 프로필 페이지로 이동
- "댓글 N개 모두 보기": 게시물 상세 페이지로 이동

### 알림 동작

- **알림 클릭:**

  - LIKE → 해당 게시물 상세로 이동
  - COMMENT → 해당 게시물 상세로 이동
- **페이지 진입 시:** 자동으로 모든 알림 읽음 처리

## 공통 컴포넌트

### 토스트 알림
```
┌──────────────────────────────┐
│ (성공) 게시물이 업로드되었습니다 │  ← 우측 상단, 3초 후 자동 닫힘
└──────────────────────────────┘
```

### 로딩 스켈레톤
- 피드 카드: 아바타 원형 + 이미지 영역 + 텍스트 라인 스켈레톤
- 프로필 그리드: 3열 정사각형 스켈레톤
- 알림 아이템: 아바타 + 텍스트 라인 스켈레톤

### 빈 상태 (Empty State)
- 피드 비어있음: "아직 게시물이 없습니다. 사용자를 팔로우해보세요!"
- 프로필 게시물 없음: "아직 게시물이 없습니다."
- 알림 없음: "새로운 알림이 없습니다."

### 확인 다이얼로그
- 게시물 삭제, 댓글 삭제, 언팔로우 시 확인 다이얼로그 표시
