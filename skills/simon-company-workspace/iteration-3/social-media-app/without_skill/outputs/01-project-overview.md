# 프로젝트 개요: Photogram (인스타그램 클론 소셜 미디어 웹앱)

## 프로젝트 목표
인스타그램과 유사한 사진 기반 소셜 미디어 웹 애플리케이션을 구축한다.
사용자가 사진을 업로드하고, 다른 사용자를 팔로우하며, 좋아요/댓글 기능을 사용할 수 있는 플랫폼을 만든다.

## Phase 1 범위 (MVP)
Phase 1에서는 핵심 기능만 구현하여 최소 기능 제품(MVP)을 완성한다.

### 포함 기능
1. **사용자 인증**: 회원가입, 로그인, 로그아웃, 프로필 관리
2. **사진 업로드**: 사진 업로드 및 게시물 작성 (캡션 포함)
3. **팔로우 시스템**: 사용자 팔로우/언팔로우
4. **피드**: 팔로우한 사용자의 게시물을 최신순으로 표시
5. **좋아요**: 게시물에 좋아요/좋아요 취소
6. **댓글**: 게시물에 댓글 작성/삭제
7. **알림**: 팔로우, 좋아요, 댓글 발생 시 알림

### 제외 기능 (Phase 2 이후)
- 스토리 기능
- DM(다이렉트 메시지)
- 해시태그 검색
- 위치 태그
- 릴스/동영상
- 추천 알고리즘 (Phase 1은 최신순만)
- 신고/차단 기능
- 이미지 필터

## 대상 사용자
- 사진을 공유하고 소통하고 싶은 일반 사용자
- 웹 브라우저 환경에서 접근

## 성공 기준
- 사용자가 가입 후 3분 이내에 첫 게시물을 올릴 수 있어야 한다
- 피드 로딩 시간 2초 이내
- 모바일 및 데스크톱 반응형 지원
