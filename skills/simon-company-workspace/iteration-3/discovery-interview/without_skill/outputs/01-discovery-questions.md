# 고객 데이터 활용 프로젝트 - 디스커버리 인터뷰 질문지

## 목적
고객 데이터 활용에 대한 요구사항을 구체화하기 위해, 이해관계자에게 확인해야 할 핵심 질문들을 정리합니다.

---

## 1. 현재 데이터 현황 파악

### 1.1 데이터 종류 및 규모
- 현재 어떤 종류의 고객 데이터를 수집하고 있나요? (예: 행동 로그, 구매 이력, 인구통계, 앱 사용 패턴 등)
- 데이터는 어디에 저장되어 있나요? (예: RDB, 데이터 웨어하우스, 로그 스토리지 등)
- 대략적인 데이터 규모는 어느 정도인가요? (일일 생성량, 전체 누적량)
- 데이터의 보존 기간 정책이 있나요?

### 1.2 데이터 품질
- 현재 데이터의 정합성이나 품질에 대해 알려진 이슈가 있나요?
- 데이터가 실시간으로 수집되나요, 배치로 수집되나요?
- 개인정보 처리(마스킹, 암호화, 동의 관리)는 어떻게 되어 있나요?

### 1.3 기존 데이터 활용
- 현재 데이터를 활용하는 방식이 있다면 무엇인가요? (리포트, 대시보드 등)
- 데이터 분석을 담당하는 팀이나 인력이 있나요?
- 현재 사용 중인 분석 도구가 있나요? (예: Looker, Metabase, Redash, 자체 도구 등)

---

## 2. 내부 사용자(직원) 관련 질문

### 2.1 대상 파악
- 데이터를 볼 필요가 있는 직원은 어떤 직군인가요? (마케팅, 영업, CS, 경영진, PM 등)
- 각 직군별로 필요로 하는 데이터의 종류와 깊이가 다를 것 같은데, 구체적으로 어떤 차이가 있나요?
- 기술적 역량 수준은 어떤가요? (SQL 가능 여부, 데이터 리터러시 수준)

### 2.2 사용 시나리오
- "데이터를 쉽게 본다"는 것이 구체적으로 어떤 상황을 말씀하시는 건가요?
  - 일별/주별 핵심 지표 모니터링?
  - 특정 고객 세그먼트 분석?
  - 캠페인 성과 확인?
  - 실시간 현황 파악?
- 현재 데이터를 보기 어려운 이유가 무엇인가요? (접근 권한, 도구 부재, 데이터 흩어짐 등)
- 데이터를 확인하기 위해 현재 어떤 과정을 거치나요? (개발자에게 요청, 직접 DB 조회 등)

### 2.3 의사결정 지원
- 데이터 기반으로 내리고 싶은 의사결정에는 어떤 것들이 있나요?
- 정기적으로 필요한 리포트가 있나요? 현재 어떻게 만들고 있나요?
- 알림이나 경고가 필요한 지표가 있나요? (예: 이탈률 급등, 매출 변동 등)

---

## 3. 고객 경험 관련 질문

### 3.1 현재 고객 경험
- "더 좋은 경험"이라고 하셨는데, 현재 고객이 겪고 있는 불편이나 아쉬운 점이 있나요?
- 고객으로부터 받은 피드백 중 데이터 활용으로 해결할 수 있어 보이는 것이 있나요?
- 경쟁사가 고객에게 제공하는 것 중 우리가 못하고 있는 것이 있나요?

### 3.2 개인화
- 고객별 맞춤형 콘텐츠/추천을 제공하고 싶은 생각이 있나요?
- 고객 세그먼트별로 다른 경험을 제공하는 것에 관심이 있나요?
- 현재 고객에게 어떤 채널로 커뮤니케이션하고 있나요? (앱 푸시, 이메일, SMS 등)

### 3.3 고객 여정
- 고객의 가입부터 이탈까지의 여정을 파악하고 있나요?
- 이탈 방지나 재방문 유도를 위한 활동을 하고 있나요?
- 고객 만족도를 측정하는 방식이 있나요?

---

## 4. 비즈니스 목표 및 제약

### 4.1 목표
- 이 프로젝트를 통해 달성하고 싶은 가장 중요한 비즈니스 목표는 무엇인가요?
- 성공 지표를 정한다면 어떤 것이 될까요?
- 단기(3개월)와 중기(6-12개월) 목표를 나눠서 생각한다면?

### 4.2 제약 조건
- 투입 가능한 개발 인력은 어느 정도인가요?
- 예산 범위가 있나요? (외부 솔루션 도입 vs 자체 개발)
- 타임라인에 대한 기대치가 있나요?
- 데이터 관련 법적 규제(개인정보보호법, GDPR 등)에 대해 검토된 것이 있나요?

### 4.3 기술 환경
- 현재 사용 중인 기술 스택은 무엇인가요?
- 클라우드 인프라를 사용하고 있나요? (AWS, GCP 등)
- 기존에 데이터 파이프라인이 구축되어 있나요?
- CI/CD 환경은 어떻게 되어 있나요?

---

## 5. 우선순위 확인

- 다음 중 가장 시급한 것은 무엇인가요?
  1. 직원용 데이터 대시보드 / 분석 도구
  2. 고객 개인화 경험 개선
  3. 데이터 인프라 정비 (수집/저장/파이프라인)
  4. 자동화된 인사이트 / 알림 시스템
- 가장 먼저 효과를 보고 싶은 영역은 어디인가요?
- "Quick Win"으로 빠르게 보여줄 수 있는 성과가 필요한가요?
