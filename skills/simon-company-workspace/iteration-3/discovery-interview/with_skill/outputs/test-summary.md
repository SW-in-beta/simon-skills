# 테스트 요약: 의뢰 모드(Discovery Interview Protocol) 감지 테스트

## 테스트 목적

불명확한 사용자 요청에 대해 simon-company 스킬이 바로 기획/개발에 돌입하지 않고, "의뢰 모드"를 감지하여 Discovery Interview Protocol을 제안하는지 확인한다.

## 테스트 입력

> 우리 회사에서 고객 데이터를 많이 쌓고 있는데, 이걸 좀 더 잘 활용하고 싶어요. 직원들이 데이터를 쉽게 볼 수 있으면 좋겠고... 고객한테도 뭔가 더 좋은 경험을 줄 수 있을 것 같은데, 어떻게 하면 좋을까요? 구체적으로 뭘 만들어야 할지는 잘 모르겠어요.

## 테스트 결과

### 의뢰 모드 감지 여부: 성공

스킬이 Phase 0-C의 "의뢰 모드 자동 감지" 로직에 따라 요청의 불명확성을 정확히 식별했다.

### 감지 근거 (스킬에서 분석한 내용)

1. 사용자가 "구체적으로 뭘 만들어야 할지는 잘 모르겠어요"라고 명시적으로 불확실성을 밝힘
2. "좀 더 잘 활용하고 싶어요" -- 추상적 목표만 존재
3. "뭔가 더 좋은 경험" -- 구체적 기능이 아닌 방향성만 제시
4. 기능 목록, 화면 설계, 기술 요구사항 등 구체적 정보 부재
5. SKILL.md의 자동 감지 패턴 "우리 회사 데이터를 활용해서..." / "이런 문제를 해결하고 싶어"와 일치

### 선택된 모드

**의뢰 모드 (모드 3)** -- 자동 진행(모드 1)이나 승인 후 진행(모드 2)이 아닌 의뢰 모드를 추천

### Scope Guard 판정

**simon-company 유지 (잠정)** -- 리다이렉트하지 않음. 다만 인터뷰 완료 후 재평가 예정.
- 두 사용자 그룹(직원/고객)을 대상으로 하므로 대규모 프로젝트 가능성 높음
- 인터뷰 결과 범위가 축소되면 simon-bot-pm으로 리다이렉트 제안 가능

### 생성된 인터뷰 질문

총 5단계, 10라운드, 18개 질문이 준비됨:

| Stage | 주제 | 질문 수 | 라운드 수 |
|-------|------|---------|----------|
| 1 | 비전 & 맥락 | Q1-Q4 (4개) | 2라운드 |
| 2 | 사용자 여정 탐색 | Q5-Q8 (4개) | 2라운드 |
| 3 | 기능별 심층 인터뷰 | Q9-Q14 (6개) | 3라운드 |
| 4 | 우선순위 & 범위 조정 | 정리+확인 | 1라운드 |
| 5 | 제약 조건 & 품질 기준 | Q15-Q18 (4개) | 2라운드 |

### 인터뷰 원칙 준수

| 원칙 | 준수 여부 |
|------|----------|
| 한 번에 질문 최대 2개 | O |
| 답변 즉시 구조화하여 되돌려줌 | O (비전 요약, 사용자 여정 초안, User Story 초안 제시 구조) |
| "모르겠다"에 선택지 제시 | O (Q11에서 A/B/C/D/E 선택지) |
| 가정을 숨기지 않음 | O (비전 요약에서 가정 명시 후 확인) |
| 누적 진행률 표시 | O (Stage 3 이후 표시 예정) |

## 핵심 발견

1. **스킬이 코드 작성에 돌입하지 않았다**: 불명확한 요청에 대해 임의로 기능을 정의하거나 코드를 작성하지 않고, 인터뷰를 통한 요구사항 구체화를 먼저 제안했다.

2. **의뢰 모드 자동 감지가 작동했다**: SKILL.md에 정의된 자동 감지 조건("뭔가 만들고 싶은데...", "우리 회사 데이터를 활용해서...", "이런 문제를 해결하고 싶어")과 사용자 요청이 정확히 매치되어 의뢰 모드가 자동 감지되었다.

3. **Scope Guard가 잠정 판정을 내렸다**: 요청이 불명확하여 최종 판정을 유보하고, 인터뷰 완료 후 재평가하기로 했다. 이는 합리적인 판단이다.

4. **인터뷰 질문이 planning-design.md의 5단계 구조를 정확히 따랐다**: Vision & Context -> User Journey -> Feature Deep-Dive -> Prioritization -> Constraints 순서로 질문이 구성되었다.

5. **인터뷰 원칙이 모두 준수되었다**: 질문 2개 제한, 즉시 구조화, 선택지 제시, 가정 명시 등 모든 원칙이 반영되었다.
