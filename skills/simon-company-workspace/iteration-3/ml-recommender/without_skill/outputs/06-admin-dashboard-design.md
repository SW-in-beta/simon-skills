# 관리자 대시보드 설계

## 1. 대시보드 개요

추천 시스템의 성능을 실시간으로 모니터링하고, 모델을 관리할 수 있는 관리자 전용 웹 대시보드이다.

**기술 스택:**
- Frontend: React 18 + TypeScript
- 차트: Recharts
- UI 컴포넌트: shadcn/ui + Tailwind CSS
- 상태 관리: TanStack Query (서버 상태)
- 라우팅: React Router v6

## 2. 페이지 구성

### 2.1 전체 메뉴 구조

```
관리자 대시보드
├── 대시보드 홈 (Overview)
├── 성능 지표 (Metrics)
│   ├── 추천 성능
│   └── 시스템 성능
├── 모델 관리 (Models)
│   ├── 모델 목록
│   └── 모델 상세
├── 상품 분석 (Items)
│   ├── 인기 상품
│   └── 추천 분포
└── 설정 (Settings)
```

## 3. 페이지별 상세 설계

**KPI 카드 구성요소:**
- 현재 값 (큰 폰트)
- 이전 기간 대비 변화량 (화살표 + 색상: 증가=초록, 감소=빨강)
- 미니 스파크라인 차트

### 3.2 성능 지표 (Metrics) - 추천 성능

**차트 목록:**

| 차트 | 유형 | 설명 |
|------|------|------|
| CTR 추이 | Line Chart | 시간별/일별 클릭률 변화 |
| 전환율 추이 | Line Chart | 시간별/일별 전환율 변화 |
| 커버리지 추이 | Area Chart | 추천된 고유 상품 비율 변화 |
| 다양성 지수 | Line Chart | 추천 목록 내 다양성 변화 |
| 이벤트 분포 | Stacked Bar | view/click/cart/purchase 비율 |
| 추천 소스 분포 | Pie Chart | user_cf/item_cf/als/popularity 비율 |
| Precision/Recall | Line Chart (dual axis) | 오프라인 평가 지표 추이 |

**필터 옵션:**
- 기간: 1일, 7일, 30일, 90일, 커스텀
- 집계 단위: 시간별, 일별
- 모델 버전: 드롭다운 선택

### 3.3 성능 지표 (Metrics) - 시스템 성능

| 차트 | 유형 | 설명 |
|------|------|------|
| API 응답 시간 | Line Chart (p50, p95, p99) | 응답 시간 분포 |
| 요청량 (RPS) | Area Chart | 초당 요청 수 |
| 캐시 히트율 | Line Chart | Redis 캐시 효율성 |
| 에러율 | Line Chart | HTTP 에러 비율 |
| 콜드 스타트 비율 | Line Chart | 인기도 기반 폴백 비율 |

### 3.4 모델 관리 (Models) - 모델 목록

**테이블 컬럼:**

| 컬럼 | 설명 |
|------|------|
| 모델 타입 | user_cf / item_cf / als |
| 버전 | 타임스탬프 기반 버전 |
| 상태 | active / completed / training / failed |
| Precision@10 | 오프라인 평가 |
| NDCG@10 | 오프라인 평가 |
| 학습 소요 시간 | 초 단위 |
| 학습 완료 시각 | datetime |
| 액션 | [상세보기] [활성화] [아카이브] |

**기능:**
- 상태별 필터링
- 모델 타입별 필터링
- 수동 재학습 버튼 (확인 다이얼로그 포함)

### 3.6 상품 분석 (Items) - 인기 상품

**테이블 (정렬 가능):**

| 컬럼 | 설명 |
|------|------|
| 순위 | 추천 빈도 순위 |
| 상품 ID | 외부 상품 ID |
| 상품명 | 상품 이름 |
| 카테고리 | 카테고리 |
| 추천 횟수 | 해당 기간 추천된 횟수 |
| 클릭 수 | 추천 후 클릭 수 |
| CTR | 상품별 클릭률 |
| 전환 수 | 추천 후 전환 수 |
| 전환율 | 상품별 전환율 |

**차트:**
- 상위 20개 상품 추천 빈도 (Horizontal Bar)
- 카테고리별 추천 분포 (Treemap)

### 3.7 상품 분석 (Items) - 추천 분포

**시각화:**
- 상품별 추천 빈도 히스토그램: 롱테일 분포 확인
- Gini 계수 표시: 추천 집중도/공정성 측정
- 카테고리별 CTR 비교 (Grouped Bar Chart)

## 4. 공통 UI 컴포넌트

### 4.1 기간 선택기 (Period Selector)
- 프리셋: 1일, 7일, 30일, 90일
- 커스텀 날짜 범위 선택 (Date Range Picker)
- 선택 시 모든 차트/테이블 자동 갱신

### 4.2 자동 새로고침
- 기본: 5분마다 데이터 갱신
- 토글로 자동 새로고침 ON/OFF
- 마지막 갱신 시각 표시

### 4.3 데이터 내보내기
- CSV 다운로드 버튼 (테이블 데이터)
- 차트 이미지 다운로드 (PNG)

### 4.4 반응형 레이아웃
- 최소 지원 해상도: 1280x720
- 2컬럼 그리드 기본, 좁은 화면에서 1컬럼

## 5. 인증 및 권한

### 5.1 로그인
- JWT 기반 인증
- 로그인 페이지: 이메일 + 비밀번호

### 5.2 역할

| 역할 | 권한 |
|------|------|
| viewer | 대시보드 조회만 가능 |
| operator | 조회 + 모델 재학습 트리거 |
| admin | 전체 권한 (설정 변경 포함) |

## 6. 알림 (Notifications)

대시보드 내 알림 벨 아이콘으로 표시:

| 알림 조건 | 심각도 | 설명 |
|-----------|--------|------|
| CTR이 기준 이하로 하락 | Warning | CTR < 3% (최근 24시간) |
| 모델 학습 실패 | Error | 배치 학습 실패 |
| API 응답 시간 초과 | Warning | p95 > 500ms (최근 1시간) |
| 캐시 히트율 급락 | Warning | 히트율 < 50% |
| 모델 학습 완료 | Info | 새 모델 배포 |
| 커버리지 하락 | Warning | 커버리지 < 30% |

## 7. 와이어프레임 참조

각 페이지의 상세 와이어프레임은 위 섹션의 ASCII 레이아웃을 참조한다. 실제 구현 시 Figma로 상세 디자인을 진행할 수 있다.
